# LUMÉ v5 — AI Memory Film Maker

## Deploy to Vercel (5 minutes, free)
1. Go to vercel.com → log in
2. Click "Add New" → "Project"  
3. Drag the entire `lume-v5` folder onto the dashboard
4. Click Deploy → live in 30 seconds

OR connect GitHub for auto-deploy on every update.

## Files
- index.html — Complete app (~95KB)
- manifest.json — PWA installable
- sw.js — Offline support
- icon.svg — App icon
- vercel.json — Deploy config

## Music Fix
50 CC0 tracks embedded with direct Pixabay CDN URLs.
Click the ▶ button directly on any track row to play.
The browser requires a real user click to start audio — this is working correctly.
